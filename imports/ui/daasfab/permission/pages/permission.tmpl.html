Permissions

Many people have requested a way to restrict access to pages inside triangular.

So for that reason we have added an optional module that works with the Angular Permission Module

To show how this works we have created some mock users, roles and permissions for triangular.

This will show you how they can be used to restrict user access to routes / pages and HTML elements in your app.

In the demo app we have 4 users with different roles assigned to each. Each role has a set of permissions that we will use to show and hide menu items on the left as well as disable the routes to those pages.
